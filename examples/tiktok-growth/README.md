# Example: TikTok Growth Engine

Real-world example of ShareClaw used to grow a TikTok account from 200 to 10K+ views per post.

## Setup
- 2 AI agents sharing state (Heisenberg + Rutherford)
- One generates content, one analyzes results
- Both read/write to shared_brain.md
- Autonomous loop runs every 12 hours

## Variables Tested (in order)
1. Hook style: motivational → ragebait → challenge
2. Video format: study desk → veggie cutting → vlog montage
3. Audio: Chopin → CML Lo-Fi → autoAddMusic
4. Posting time: morning → afternoon → evening → late night
5. Caption style: short → story → question → controversial

## Key Results
- Ragebait hooks outperformed motivational by 2x
- Short hooks (1-2 lines) beat long hooks (8 lines) every time
- 4 hashtags from competitor data > generic 12 hashtags
- Organic engagement (manual likes/comments) boosted algorithm trust

See `shared_brain.md` and `execution.md` in this directory for the full state.
